    It is used to define a hyperlink ,allowing users to click and navigate to another page,section,or resource,such as a file or an external URL.

  8. br tag

    It is used to insert a line break, meaning it forces the text following it to start on the next line.It simply moves the text down to the next line without adding any extra spacing.
